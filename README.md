
# 📰 Clone TabNews - curso.dev

Este repositório contém um clone do [TabNews](https://www.tabnews.com.br), desenvolvido com o propósito de estudo e prática. O projeto foi construído como parte do acompanhamento do [curso.dev](https://curso.dev), ministrado por Felipe Deschamps.

Mais do que replicar funcionalidades, este espaço serve como um laboratório contínuo de aprendizado em desenvolvimento web. É aqui que realizo testes de infraestrutura, implemento melhorias e consolido os conceitos técnicos abordados ao longo do curso.

---

## 💻 Sobre o Projeto

O TabNews é um espaço para quem trabalha com tecnologia encontrar conteúdos que possuem valor concreto. O objetivo deste clone é recriar essa experiência, entendendo os desafios reais de construção de uma aplicação de ponta a ponta.

### 🎯 Objetivos de Aprendizado:
* Entender e aplicar fundamentos sólidos de programação.
* Configurar e gerenciar infraestrutura (Docker, Banco de Dados).
* Criar APIs robustas e conectá-las ao Front-end.
* Aplicar boas práticas de versionamento e deploy.

## 🚀 Como Executar o Projeto Localmente

Pré-requisitos: Você precisa ter o [Node.js](https://nodejs.org/) e o [Docker](https://www.docker.com/) instalados na sua máquina.

1. Clone este repositório:
   ```bash
   git clone git@github.com:sandroccosta/nome-do-seu-repositorio.git
   ```

2. Acesse a pasta do projeto:
   ```bash
   cd nome-do-seu-repositorio
   ```

3. Instale as dependências:
   ```bash
   npm install
   ```

4. Suba os serviços de infraestrutura (Banco de Dados) com o Docker:
   ```bash
   docker compose up -d
   ```

5. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

A aplicação estará disponível em `http://localhost:3000`.
